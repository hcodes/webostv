# Library version
A source code for library version 1.2.4.

## Install
```bash
npm i --save-dev webostvjs
```
or
```bash
yarn add webostvjs
```

## Usage
```javascript
import 'webostvjs/webOSTV';
import 'webostvjs/webOSTV-dev';
```
